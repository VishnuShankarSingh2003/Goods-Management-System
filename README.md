## Quick Installation

    git clone https://github.com/VishnuShankarSingh2003/Goods-Management-System.git

    cd Goods_Management_System
    
### Composer

    composer update
 
    
 ### For Migration table in database [Create database name as ```store```]
 
    php artisan migrate
    
### Server ON ```url: http://127.0.0.1:8000/```

    php artisan serve
